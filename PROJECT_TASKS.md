# Задачи проекта

## Позже: контент для доверия коллекционеров

- [ ] Подготовить расширенную биографию Lili Miller и творческое CV на русском, английском и китайском.
- [ ] Собрать хронологию выставок с городами, площадками, датами и фотографиями экспозиции.
- [ ] Добавить публикации, интервью, награды и ссылки на внешние материалы.
- [ ] Описать сертификат подлинности, маркировку OOAK и состав комплекта каждой работы.
- [ ] Подготовить FAQ о частных заказах, сроках, оплате, упаковке и международной доставке.
- [ ] Добавить рекомендации по хранению и уходу за фарфоровыми куклами.
- [ ] Собрать отзывы коллекционеров и фотографии работ в коллекциях — только с разрешения владельцев.
- [ ] Подготовить страницу процесса: эскиз, скульптура, отливка, обжиг, роспись, костюм и финальная сборка.

Эти материалы следует публиковать только после редакторской проверки фактов и профессиональной вычитки переводов.
